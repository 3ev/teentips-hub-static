# Teen Tips Hubs Static site

This is a Sass/HTML implementation of the Teen Tips Hub design

## Installation

### 1. Install dependancies

```bash
yarn
```

### 2. Build static files


```bash
npm run build
```

## Develop locally

### 1. Run dev command

```bash
npm run dev
```

### 2. Run a local server

```bash
npm run server
```
